# Treinamento Malu — Skymarine

Pacote pronto para publicar no GitHub Pages.

## Como subir no GitHub

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste ZIP para a raiz do repositório.
3. O arquivo principal já está nomeado como `index.html`.
4. No GitHub, acesse **Settings > Pages**.
5. Em **Build and deployment**, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Salve e aguarde o GitHub gerar o link.

## Arquivos incluídos

- `index.html`: página principal do treinamento para GitHub Pages.
- `treinamento_malu_skymarine.html`: cópia com o nome original do arquivo.
- `.nojekyll`: evita processamento Jekyll no GitHub Pages.
- `README_GITHUB.md`: instruções de publicação.

## Observação

Este protótipo HTML é autônomo e não depende de imagens externas, CSS externo, JavaScript externo ou banco de dados. As funções de autenticação, logs, IP, certificado validável e dashboard são demonstrativas/offline.
